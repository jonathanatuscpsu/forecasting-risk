Definition of diff():

The diff function computes the difference between pairs of consecutive elements of a numeric vector.

In R we can use the diff(x,lag,diff) function for differencing a time series, which requires 3 arguments:

- x (the data)
- lag (the lag at which to difference)
- differences/diff (the order of differencing).

Note: The second argument is the lag, which is the number of periods, and the third argument is differences, which is the order of the difference (e.g. how many times diff() is called).

Tips

  • First-differencing a time series will remove a linear trend (i.e., differences=1).
  • First-differencing a time series at a lag equal to the period will remove a seasonal trend (e.g., set lag=12 for monthly data).
  • Twice-differencing will remove a quadratic trend (i.e., differences=2).

Basics of Mathematical Notation for Time Series (Time Lag)

\(y_{t-j}\): jth lag of y

\(y_{t-j}\): the value of y in time period t-j (the value of j periods earlier).

  • \(\Delta y_t=y_t - y_{t-1}\)
  • \(\Delta y_{t-1} = y_{t-1} - y_{t-2}\)
  • \(\Delta y_{t-j}=y_{t-j}-y_{t-j-1}\)

Each time we lag a time series one period we lose an observation.

Each time we difference a time series we lose an observation.
